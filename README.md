# Node Js REST API with Authentication & Role-Based Access (Authorization) with a simple frontend UI

## This is a Notes Application developed using HTML, Bootstrap, Javascript, Node Js (Express Js)

## Table of Contents
- [Introduction](#introduction)
- [Features](#features)
- [Installation](#installation)
- [Usage](#usage)


## Introduction

This is a notes application, where users can create, read and delete notes.


## Features

* Admin and Users can Register to the application
* Admin can access users and notes, i.e., admin can create, read, delete users
* Users can access notes, i.e., users can create, read, delete notes

## Installation

* To Installation the project, follow the steps below
1. Clone the Repository
2. Install dependencies
    ```cmd
        cd backend
        npm install
    ```

## Usage

* To start the project, run the following command (make sure you add .env file)
```cmd
    npm start
```
* Refer [Documentation](https://documenter.getpostman.com/view/49939023/2sB3WtrJdn) 

## Open UI

* To open the UI of the application
* Open FrontEnd Folder and run the index.html file in web browser
